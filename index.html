D&D Expert Adventure · TSR 9114 · 1984 · Levels 5–10

Quest for the
Heartstone

by Michael L. Gray  ·  XL-1

The king is dead. A magical stone lies hidden in the frozen mountains — and the Master Thief Dahnakriss may have beaten you to it. A wilderness and dungeon adventure through marsh, glacier, and four dungeon levels deep beneath the Mountains of Ice.

DM Tools

🌫️ Figma Fog-of-War Plugin

Upload any map to Figma, select it, and run this plugin to cover it with a grid of removable fog tiles. Delete tiles as players explore. Download manifest.json — install via Figma → Plugins → Development → Import from manifest.

🗺️ Owlbear Rodeo

Free browser-based VTT. Download a map image, drag it into a new Owlbear Rodeo room, and share the room link in Discord. Built-in fog of war, token movement, no accounts needed for players.

🎲 Rules Reference

This module uses BECMI D&D Expert rules (compatible with Basic/Expert sets). All saving throws and class abilities follow the Expert rulebook. Character level range: 5–10.

📋 Character Sheets

All 19 pre-rolled characters have interactive sheets with editable HP, 20-slot inventory, treasure tracker, and spell memorization. Open any sheet in a browser or upload to Google Drive to edit as a Google Doc.
