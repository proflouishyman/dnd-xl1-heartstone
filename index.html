Live Shared Maps — Real-Time Fog

All six maps have built-in real-time sync. Players see your reveals instantly in their browser — no Figma, no account, no install.

Step 1 — Open a map above

Click any map in the grid above. You're in DM mode by default — fog tiles are clickable, state saves to your browser automatically.

Step 2 — Start a Live Session

Click "Start Session" in the toolbar. A share link appears instantly. The link stays stable for that map so you can reuse it every session.

Step 3 — Share the Link

Click "Copy Link" and drop it in Discord. Players open it in their browser — they're in read-only mode and see exactly what you reveal, live.

During Play

Click or drag tiles to reveal rooms. Players see each reveal the moment you click. Use Reset to re-fog everything between sessions. The player count shows who's connected.

🎲 Rules Reference

This module uses BECMI D&D Expert rules (compatible with Basic/Expert sets). All saving throws and class abilities follow the Expert rulebook. Character level range: 5–10.

📋 Character Sheets

All 19 pre-rolled characters have interactive sheets with editable HP, 20-slot inventory, treasure tracker, and spell memorization. Open any sheet in a browser or upload to Google Drive to edit as a Google Doc.
